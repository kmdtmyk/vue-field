https://kmdtmyk.github.io/vue-fields/

## Installation

```
npm install @kmdtmyk/vue-fields
```

```javascript
import VueFields from '@kmdtmyk/vue-fields'

Vue.use(VueFields)
```

or

```javascript
import {IntegerField} from '@kmdtmyk/vue-fields'

Vue.component('integer-field', IntegerField)
```
